This program is a Python Script that reads Python script which reads data from the files
CO2.csv and temperature.csv and generates a labeled, nice plot of time vs CO2
and time vs. temperature. It also takes as input an upper/lower threshold, and generates
chart of the CO2 emissions of all countries with per capita
